# Secure Networking Website Project

## Overview
This project is a secure, optimized, and monitored website built to demonstrate real-world networking concepts, including web security, performance optimization, and traffic monitoring.

## Features Implemented

### Security
- HTTPS enabled via GitHub Pages
- XSS protection using input sanitization
- Basic brute-force protection using attempt limiting

### Performance
- Lazy loading images for faster load time
- Lightweight static website structure

### Monitoring
- Google Analytics integration for traffic tracking

## Deployment
